[How-To] Run Android on your Windows PC with Android SDK

In this Tech Savvy world, Android has become the most popular Mobile Operating System along with Apple's iOS somewhat. The main features with which Android has made its mark on this techno world is its compatibility as well as its speed and of course, it's awesome user interface.

Android is now becoming popular among the application developers as well as some Linux enthusiasts. As the Android is based upon Linux. It's a great Open Source operating system which lures more and more programmers and developers to it.

In this article, We are going to represent the image tutorial on How to run Android on your Windows enabled machine. If you want to experience the Android, but at the same time, if you don't want to shell out some bucks for it, try this trick to experience Android on your machine. (This is also a good feature of Android.) And if you want to develop applications, it is must to test your applications.

There exist mainly two ways for this purpose.
  1. Using Android SDK
  2. By Installing Android on your Portable Drives (e.g, Pen Drive, Portable HDD etc.)
In this article, we are going to focus on the first one, which is using Android SDK. Though it's more complicated, it's worth trying as it installs and runs along the native OS of your machine. The other trick is to install Android on the portable drives. This trick is very safe and easy to install and use, but we will discuss it later.

Installing Android SDK

Installing Android SDK is easy. First download Android SDK according to your machine OS. We are using SDK for Windows. You can download Android SDK from this link.

Download Android SDK

We have downloaded android-sdk_r12-windows.zip for this tutorial.You are free to download any of it. We recommend .zip instead of .exe.

Now, extract the downloaded zip archive to your desired location.

After extracting it, open SDKManager.exe. It will open the window like this.


Now, let it search for available resources. After the search is completed, it will display the window like this. 



Here, you will have to select the version of Android that you want to install. We have used Android 2.2 here. The version that you don't want to install, check reject button.


After clicking Install, it will start downloading the resources.



If Download don't start, goto Settings and uncheck "Force https://...sources to be fetched using http://... ".




Now, after installation is complete, we will go to Setup process.


Setup Android SDK

To setup Android, go to "Virtual Devices" Tab and click "New.."
After clicking "New..", you will get the following window.

Select the options as shown in Window.




Now click on "Create AVD", the program will seem to be hanged up at this time. Wait until the following window appears.




Now the created AVD will be shown up in the "Virtual Devices" tab. Select it and press "Start..".




After clcking "Start..", you will be prompted to select screen resolution.




After clicking "Launch", It will hang for some time and then the Android will start functioning normally.

You can do everything that you do with a phone.




You can see menu and install apps.




Now you are done! Browse the web with Android 2.2 and test your apps.
